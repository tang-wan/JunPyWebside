++++++++++++++
.. ========================
.. --------------------------------
.. ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. caution:: 
   Because JunPy is written by python and C++, 
   we suggest running Junpy under linux envirement and anaconda will be more convenient to compile

Install for linux
========================

#. Copy Junpy file in to system
#. Create envirement
#. export CC=gcc
#. export CXX=g++
#. pip install .
